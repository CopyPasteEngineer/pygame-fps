### Install
ติดตั้งตัวรันภาษา Python version 3.7.X จาก [www.python.org/downloads/](https://www.python.org/downloads/)

ติดตั้ง modules เสริม
```
pip install pygame
pip install numpy
```

### วิธีรัน
เข้าไปที่ folder ของ part ที่สนใจ แล้วรันคำสั่งใน CMD:
```
python main.py
```
